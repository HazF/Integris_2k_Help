Taking a Screen Snapshot

The Integris 2000 has a snapshot feature that allows you to take screen shots when a set of values or the display is showing something you would like to save for later.

Information: Images are saved in JPEG (.jpg) format in the Integris-2000\AA_SCREEN_CAPTURE\ folder. File names start with an 'SC' followed by the date and then the time (e.g.SC20151125_084539_695.jpg

Click on the camera icon to take a screen shot. The icon briefly displays a number '1' to indicate a screenshot has been taken. (See Fig 1.1 )

Click on the 'eye' icon to select a previously saved screen shot that you want to display.
