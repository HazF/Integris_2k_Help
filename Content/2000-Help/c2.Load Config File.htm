Loading a Test Configuration File

Location  
GUI or Keyboard Test Config icon at the top of the Main Window,
Classic menu Emulate > LoadConfig
Modern menu Emulate > Load Config

 

Radio Head testing is accomplished by transmitting a test sequence that is matched to the equipment being tested. These tests are defined by a test configuration file.

 

Test Configuration File select icon

 

 

To load a configuration file:

  1. click on the Test Config icon at top left of the main screen. (See Test Configuration File select icon)
  2. Select the required configuration file from the list. (See Selecting a test Configuration File)

 

Click here for more information on Configuration Files

 

NOTE: From time to time you may need to update or add Configuration files. These are located in the C:\Integris-2000\Configs\ folder.
