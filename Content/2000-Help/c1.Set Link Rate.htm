Link Rate icon

 

Link status

 

To set the link rate:

  1. Click on the Rate icon in the Status panel at top left of the main window. A drop down menu will appear. The currently selected rate is highlighted
  2. Click on the required link rate. (See Link Rate icon)

The Link text on the icon will display the words "Active" in green if the local SFP recognizes an active CPRI link, or a red error message if a problem exists (See Integris Link Status Reports below)
