At the end of each test, Integris 2000 creates a set of result values that were accumulated during the test. As detailed in the procedures for the individual tests, these can be saved and accumulated into reports.

  • Click on the Report menu to view options related to report preparation.
  • Optionally right-click on the report view to open further options (See Right-click report options)

Right-click report options

 

 

Information: By default, individual results are saved with a "DATE_TIME_Report" name format (eg. 151204_135518_Report) indicating that the test was completed on the 4th December 2015 at 1:55PM).

 

Modern Report menu

 

Classic Report menu

 

View (Hot key, Ctrl + R): Opens the current report and shows any test results that have been included so far.

Select: Allows you to open a previously saved report and any test results it may contain. Results from any new tests will be added to this report. Clicking on any of the test results in the opened report expands its contents for viewing.

New: Creates a new empty report using the current date and time to form the filename.

NOTE: It is a good idea to create a system for report names that gives information on the test location or conditions. For example, using a naming convention such as <frequecy>/<Bandwidth>-<Technology>-<Sector>-<Cable Color> will give a file name similar to 700/10-LTE-6-Blue

Clicking on the yellow Add to Report button after running any of the tests will add these test results to this report. In this way you can create report files with a list of individual results.

Edit: Allows you to edit the file name and add a site details along with any notes.

Clear: Clears all test results, site name and notes from the current report and sets the date field to the current date.

Export PDF: Opens an explorer window to select a location for the exported Portable Document Format (PDF) file. After saving the report is opened for viewing. (See Figure 1.1. )

Information: See Preferences for export to PDF format options

Additional Right-Click Options

Rename: Allows you to rename the currently highlighted test result

Delete: Requests confirmation to delete the currently highlighted test report

Edit: Allows you to edit the current report (not to be confused with the test result) details along with any notes

Clear: Clears all test results, site name and notes from the current report and sets the date field to the current date.

Close View: Closes the currently highlighted View window

Move View: Moves this View window in relation to any other views that are currently displayed. Blue positioning borders will appear to allow you to place this view in relation to the others. (See Using Multiple Views)
