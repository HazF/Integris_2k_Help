Integris 2000 Hot Keys

 

Hot Keys Action
F1 Opens the Integris 2000 help
F5 Start / Stop the selected test
Alt+F4 Close the Integris 2000 application
CTRL + D Displays information on how to connect Integris 2000
CTRL + G Open a new graph view
CTRL + M Toggle monitoring on / off
CTRL + P Open Integris 2000 Preferences
CTRL + R View current report
CTRL + T View current test result
CTRL + Shift + L Load a saved profile
CTRL + Shift + S Save current profile
