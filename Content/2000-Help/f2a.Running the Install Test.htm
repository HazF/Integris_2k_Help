Step 1: Create a Test Report

(See also Working with Test Reports)

 

Report files contain one or more sets of results from the tests you run. When you start a test a default report is created for you, but it is a good idea to create one ahead of testing so you can edit the name and add details such as location and test team.

 

Click here to see how to create a new test report:

  1. Select the Report menu and click New. A new blank report will be created with a default format file name and the current date. Alternatively, click on Select to open a previously saved report.

  2. On the Report menu, click on Edit. In the report window that opens add site details and any notes you might want to include. You can also edit the file name to something more appropriate.

Step 2: Starting the Install Test sequence:

  1. Ensure that the selected test port corresponds to the test port connected to the radio head under test.

    Click here to see how to select the test port:

    Desktop Port Selection icon

     

    You can choose the desired test port by either:

    • Clicking on the BBU Port icon at the top left of the screen, then selecting the desired port. (See Port selection icon )
    • If using either Classic or Modern menus, open the Emulate menu and click on BBU Port. Select the desired port from the dropdown list

     

  2. Ensure that the correct link rate has been set and the Status Pane is displaying the link state as ACTIVE.

    Click here to see how to set the link rate:
    1. Click on the Rate icon in the Status Panel at top left of the main window. A drop down menu will appear. The currently selected rate is highlighted
    2. Click on the desired link rate.

     

  3. Ensure you have selected the correct configuration file for the radio head under test.

     

    Click here to see how to load a configuration file:
    • Click in the Test Config icon at the top left of the test view
    • On the Classic menu system, open the Test menu and click on Loadconfig
    • On the Modern menu system, open the Test menu and click on Load Config (See Manufacturers' Configuration Files)

     

    NOTE: Turnup tests offer the most extensive radio head testing. As well as Local & Remote SFPs, Physical & CPRI Layer information and radio head inventory they include radio head metrics like VSWR, Insertion Loss and TX / RX power.

     

    Manufacturers' Configuration Files

     

  4. Decide if you want to simulate user load using OCNS.

    Click here to see how to add OCNS:

     

    The number of users on a radio system loads the resources available and affects how the system performs.

     

    In order to simulate real-world loading, Integris 2000 includes the ability to apply Orthogonal Channel Noise Simulation (OCNS) to the last phase of the Install tests. This 'forward loads' the system and simulates user loading similar to those expected during in-service operation.

     

    1. Near the bottom of the Status Pane at the top left of the screen, click on OCNS

    2. Select the OCNS value required to simulate the desired circuit loading. (See OCNS setting)

      OCNS setting

       

  5. Decide if you want the radio head to transmit a Physical Cell identification (PCI). If so click on PCI at the bottom of the Status Panel, then enter the desired value.

    PCI Setting

Step 3: Running the Installation Test

  1. Open the Emulate menu and select New option. (See Modern Test menu and Classic Test Menu)

  2. Now click on Run from the same menu (or press the F5 key),

    Classic Test menu

     

    Modern Test menu

     

     

    Install Test Results showing Add to Report prompt

     

    By clicking on the Messages icon at the top left of the test results pane you can see information returned as the test progresses.

     

    Install Test Results messages

     

    Once the test sequence has been completed you can optionally save the test results by adding them to a report.

     

    To add to an existing or new report file:

    1. Click on the yellow Add To Report icon at the top right of the result panel.(See also Working with Test Reports)
    2. Choose a name for these results, or can accept the default name based on the date and time of the test.
    3. Once you have finished editing the name, click on OK.

    NOTE: The Remote SFP results section lists information related to the SFP fitted to the Radio Head. These results should NOT be confused with those gathered during Integris 2000 SFP Tests which refer to the Local SFP fitted to the Integris system.

     

Step 4: Exporting the Report File in Portable Document Format (PDF)

  1. Click on Report > Export PDF. An Explorer window will open allowing you to name and save the PDF file to a location of your choice.

  2. Click Save.
