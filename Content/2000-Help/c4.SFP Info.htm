Viewing SFP Information

Location  
GUI or Keyboard Link icon on Status Panel
Classic menu Not Applicable
Modern menu Not Applicable

 

You can access information on the link status by clicking on the Link icon at the top of the Status Panel to the left of the main window. This information relates to the Local SFP and link connected to the current Integris test port.

 

NOTE: Do not confuse the SFP referred to in link status reports with the SFP information in the Install tests.

  • The SFP referred to in some link state error messages relates to the SFP fitted into the Integris test port. It is referred to in the test results as the Local SFP.
  • The Integris 2000 Install tests also report information related to the SFP fitted to the RRH. This is referred to as the Remote SFP.

 

Information:all SFP vendors provide machine readable information regarding the SFP's capabilities. This information is called the Vital Product Data (VPD) and includes the SFP manufacturer, supported link rates, optical wavelength, etc. 

The Integris System can read this information and present it to the user so they can verify that the SFP is suitable for the application in which it is being used.
