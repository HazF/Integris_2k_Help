Checking the Software Version

Location  
GUI or Keyboard None
Classic menu System > Help > About
Modern menu Help > About

 

To open a window displaying basic software and version information (See Fig 1.1 ):

  1. On the Classic menu system, select the System menu

  2. Click on the Helpmenu, then on About

     

    Software Version Information

     

  3. To close the information window, click Done
