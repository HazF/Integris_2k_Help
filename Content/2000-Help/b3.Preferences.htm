There are further options available from the Preferences panel.

To open the Preferences panel (See Fig 1.1 ):

  1. Open the System menu
    • Click on Prefs on the Classic menu system, or
    • Click on Preferences on the Modern menu system
  2. Check the Prompt before discarding results checkbox to be reminded if the current report has not been saved before being overwritten
  3. Check the One Result Per Page checkbox to insert a page break between results when exporting a report to a PDF document
  4. Check the Landscape checkbox to export PDF reports in Landscape format
  5. Click on the Report Format dropdown menu and select between:
    • Basic which creates a report containing all the tests in sequence.
    • Expanded which creates one table with the tests arranged horizontally. This arrangement can make it easier to compare test results.
    • Compressed which creates one table with the tests arranged vertically. This arrangement can make it easier to compare test results, and may be more suitable where several tests are being compounded into one report..
  6. When you have completed your selections, click Done
