In addition to the installation tests, Integris 2000 can display 'digital' spectrum graphs, including power spectrum, persistence and/or waterfall graphs derived from the IQ data from the Radio head and carried over a CPRI link.

Step 1: Opening a graph & locating on screen

To display a power graph:

  1. If using the Classic menu system, open the Analyze menu.

  2. Click on the Graph menu menu and select New. (See Fig 1.1 )

     

    Opening a Graph view

  3. On the Create Graph dialog box that opens, select the graph Source.

    Integris 2000 allows you to select the graph type or a combination of graph types. Each type supports a different aspect of signal analysis. (See Fig 1.2 )

    Spectrum: The power magnitude of the input signal is displayed across the set frequency range (See also "Spectrum Graph Configuration" on page 1)

    Waterfall: Spectrogram giving a visual representation of frequency in the X axis against time in the Y axis

    Persistence: Power magnitude is plotted against frequency, with color denoting the percentage of time a plot point is placed at the current location

      

    Selecting a graph type

    Selecting the graph source

  4. Select the graph Source. You can select live CPRI data from Integris ports 1 or 2, or playback from a file. (See Fig 1.3 )

  5. Click on Ok

Step 2: Selecting the mapping information

If you have selected a live source, you must now select the file containing the mapping information to convert the signal data into a meaningful graph.

Information: Integris 2000 mapping files contain information on frequency, bandwidth and other information required to decode the graph from the CPRI stream.

To select a Mapping file:

  1. If using the Classic menus, select the Analyze menu
  2. Go to the Graph menu and click on Mapping.
  3. A list of the available mapping files will appear similar to Fig 1.2
     

    Selecting a graph source

  4. Select the desired Graph source from the list

Once the correct source and mapping file have been selected a graph view should appear on the screen.

 

Spectrum graph

Step 3: Placing a marker

Integris 2000 graphs have five optional markers that can be used to track frequency offsets and power values on a spectrum graph.

Note: Markers can only be used with Power Spectrum graphs

To place a marker on the graph:

  1. Move the mouse pointer close to the peak you want the marker to appear on.
  2. Right click and hover over the Snap Marker menu item. Select the desired marker number from the menu that appears.
  3. The marker will locate the highest amplitude peak within 130 kHz of the mouse pointer.
 

Snapping a marker onto a location

Step 4: Using the Marker Table

When enabled, the Marker Table is located below the on-screen graph(s) and displays amplitude and frequency offset values for each of the five markers.

 

Markers & the Marker Table

To enable the Markers Table using the Modern Menu system, Either:

Open the Markers menu and click Show Table. The Markers Table opens at the bottom of the screen under the graph(s)

or

  1. Open the Markers menu and click on Configure. The Markers control panel opens (See Fig 1.1 )
     

    The Markers control panel

  2. Set one marker as a reference by clicking to select it in the Ref column.

To enable the Markers Table using the Classic Menu system:

  1. Make sure the Analyze menu is selected
  2. On the Markers menu, click on Table to toggle the Marker Table on and off. The Markers Table opens at the bottom of the screen under the graph(s).
  3. Set one marker as a reference by clicking on Reference and selecting the desired marker number.

Table Terms

 
  • The Frequency and Power columns show the current values for this pointer.
  • The Δ Power column displays the difference between the power value for the reference marker (shown as a 0.000 db value) and the power value for this marker
  • The Δ Frequency displays the difference between the frequency offset of the reference marker (shown in the Δ Power column as a 0.000 db value) and the frequency offset for this marker

Closing the Graph

You can close the current graph by right clicking on it and selecting Close View

Alternatively open the Trace menu and click on All Off.
