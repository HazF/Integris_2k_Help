Using Multiple Views

Using Multiple Views

You can view multiple tests, test results or graphs at the same time. This can be useful when running a sequence of diagnostic tests on both test ports, while viewing the live graph as seen from the tower head.

Multiple tests and test reports

 

As you click to highlight each view the Status Panel alters to show the appropriate values for that screen.

Adding a View

Adding a View

A new view is added automatically when you start a new operation. For example, if you have a test report open, then selecting a new test will open a new Test view so that both the report and test views can be seen at the same time. The same would be true when opening a new graph when a test view is open.

To add a new Test view:

  1. If using the Classic menu system, select the Emulate menu
  2. From the Emulate menu, click on New (See Fig 1.2 ).

    Selecting a test type (Modern menu)

     

  3. Positioning borders will appear. (See Positioning the new view)

To add a new Report view:

  1. On the Report menu, select New.
  2. Positioning borders will appear. (See Positioning the new view)

To add a new Graph view:

  1. If using the Classic menu system, open the Analyze menu.

  2. Click on the Graph menu menu and select New. (See Fig 1.1 )

    Opening a Graph view

  3. On the Create Graph dialog box that opens:

    1. Select the graph Type (See "Selecting the Graph Type", page 1)(See Selecting the Graph Type)
    2. Select the graph Source. You can select live CPRI data from Integris port 1 or 2, or playback from a file.

      3. Selecting a graph type

      Selecting the graph source

  4. Click on Ok

Selecting / Deselecting Multi-View

Selecting / Deselecting Multi-View

When multiple views are available on the screen, a view control icon appears at top right of the GUI.

View a single screen (left) and all screens (right)

When multiple views are being displayed, an icon representing a single graph is displayed (See Fig 1.9 )

To view any of the individual graphs, double click the desired graph (or click to highlight it, then click on the single graph control icon).

To return to multiple graph view, double-click the onscreen graph (or click on the all-views icon).

Moving a View

Moving a View

You can move any of the graphs on the screen by right clicking on the graph and selecting Move View from the menu that appears. (See Moving a view)
