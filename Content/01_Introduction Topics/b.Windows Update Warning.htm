Windows Update Warning

Your Integris system is shipped with a fully configured Windows 7 installation. However, in recent months Microsoft updates have included an option to upgrade to Windows 10, called the "Get Windows 10 app".

Integris software is not compatible with Windows 10 and although all recently shipped systems have this upgrade blocked, some early systems may be vulnerable.

If you see the Windows upgrade offer, IT IS IMPORTANT NOT TO ACCEPT.

If the upgrade offer has already appeared on your system, follow these steps to remove the relevant Windows update

  1. Open Control Panel and navigate to Programs and Features.
  2. From the left pane click the View Installed Updates link.
  3. Look for these updates: KB3035583, KB2952664, and KB3021917.
  4. Right-click each update and from the context menu click Uninstall.
  5. Reboot your computer.

To block any future upgrade take the following steps:

  1. Open the Windows Control Panel and select System and Security
  2. Select Administrative Tools
  3. Double click Computer Configuration to open the Computer Configuration window.
  4. Click Policies.
  5. Click Administrative Templates.
  6. Click Windows Components.
  7. Click Windows Update.
  8. Double-click Turn off the upgrade to the latest version of Windows through Windows Update.
  9. Click Enable.
