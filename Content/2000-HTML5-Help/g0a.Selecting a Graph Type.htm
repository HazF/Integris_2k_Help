Selecting the Graph Type

Location  
GUI or Keyboard Ctrl + G
Classic menu Analyze > Graph > New
Modern menu Graph > New

 

Integris 2000 allows you to select the graph type or a combination of graph types. Each type supports a different aspect of signal analysis.

 

Spectrum: The power magnitude of the input signal is displayed across the set frequency range. (See also "Spectrum Graph Configuration" on page 1)

 

Waterfall: Spectrogram giving a visual representation of frequency in the X axis against time in the Y axis. (See also Setting Waterfall Graph Parameters

 

Persistence: Power magnitude is plotted against frequency, with color denoting the percentage of time a plot point is placed at the current location. (See also Persistence Graph Configuration)

 

Selecting a graph type

To select the Graph type:

  1. If using the Classic menus, select the Analyze menu
  2. Go to the Graph menu and click on Type.
  3. Select the desired Graph type or combination

The original view(s) will now appear with blue positioning borders to allow you to place the new graph in relation to them.

Fig 1.2 shows the left positioning bar selected which will place the new view to the left of the current graph. Fig 1.3 shows the new graph displayed to the left of the original graph.
