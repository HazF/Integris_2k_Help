Positioning Markers

Setting a precise position for a marker on Integris 2000 is simple and intuitive. You can either use the 'Snap' feature to find a peak close to the current location of the mouse pointer, or move a currently displayed marker using a slider on the Modern Markers Control Panel.

Setting a marker using the Snap feature:

  1. Move the mouse pointer close to the peak you want the marker to appear on.
  2. Right click and hover over the Snap Marker menu item. Select the desired marker number from the menu that appears.
  3. The marker will locate the highest amplitude peak within 130 kHz of the mouse pointer.

Precise Marker Positioning

Precise Marker Positioning

Once a marker has been activated, either by using the 'Snap' feature above, or by enabling it from the Markers control panel, you can adjust the marker's position on the Y (frequency) axis.

Using the Modern menu system

  1. Open on the Markers menu and click on the Configure menu option. The Markers control panel appears
  2. If it is not already active, click in the Active column next to the desired marker. A check appears in the Active box next to the marker and the marker appears on the screen.
  3. To move the pointer you can You now have a number of options:
    • Type the desired frequency into the Frequency box
    • Click on the arrows to the right or left of the Frequency box. This will move the marker left or right on the graph.
    • Click on Go Peak to move the marker to the point of greatest Amplitude.
    • Click on Next Peak to cycle the marker through the peaks on the spectrum graph.
