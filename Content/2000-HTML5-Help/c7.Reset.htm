Resetting Integris to Default Settings

Location  
GUI or Keyboard None
Classic menu System > Reset
Modern menu System > Reset

 

It is sometimes easier to reset an application to its defaults than work through a number of configuration settings.

 

To reset Integris 2000 to default settings select the System menu, and click on Reset.

 

A warning will be displayed to ensure you understand that your current configuration will be lost. The Integris 2000 application will then close and restart in the factory default configuration.
