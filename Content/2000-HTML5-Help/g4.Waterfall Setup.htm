Setting Waterfall Graph Parameters

The Waterfall graph type gives a visual representation of frequency in the X axis over time in the Y axis. It is useful for seeing small changes in signal amplitude as well as picking out intermittent interferers.

10MHz swept signal seen on spectrum and persistence graphs

 

To open a Waterfall graph:

  1. If using the Classic menus, select the Analyze menu
  2. Go to the Graph menu and click on Type.
  3. Select the Waterfall option (or in combination with another graph type)

Above the top left of the waterfall graph is a scale indicating the power values related to each color on the waterfall (See Fig 1.2 ) By hovering over any color in the range the amplitude represented is displayed.

Color bar chart showing the amplitude range in dB represented by Orange on the graph

 

You can change the way colors represent the values on the graph by clicking on the Reference Level icon on the Status Pane, then typing or using the slider to change the value. (See Changing color values)

Changing color values

 

To the right of the color chart is the current refresh value setting in milliseconds (ms). The refresh rate sets the period between lines being added to the top of the waterfall. Reducing the period has the effect of speeding the waterfall up and can be useful when looking for short period interferers.

Refresh setting drop-down

 

Click on the live Refresh icon to open a dropdown list enabling you to select a new refresh rate. (See Fig 1.3 )

When you hover the mouse over the waterfall the frequency offset at the current location is shown at the top right of the screen. (See Fig 1.3 )

Frequency offset at the current mouse position

 

Waterfall Graph Right Click Options

By right clicking anywhere on the graph you can open a menu of options. (See Fig 1.5 )

 

Menu Item Function
Grid

Enable this option to superimpose a grid onto the graph
Type

Opens a menu displaying a range of graph types or combinations of graph types, to display
Close View

Click to close the currently selected graph. (Only displayed if multiple views are being displayed). See Using Multiple Views
Move View

Click to move the current graph on the screen in relation to other views. (Only displayed if multiple views are being displayed). See Using Multiple Views
