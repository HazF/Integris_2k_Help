Spectrum Graph Configuration

Spectrum graphs plot the power magnitude of the input signal in decibels (dBs) on the Y axis against frequency in the X axis.

10MHz swept signal seen on spectrum and persistence graphs

 

To open a spectrum graph:

  1. If using the Classic menus, select the Analyze menu
  2. Go to the Graph menu and click on Type.
  3. Select the Spectrum option (or a combination with another graph type) at the same time selecting the Graph Source. (See also Selecting a Graph Source & Mapping)

Adding Display Barriers to the Graph

Barriers mark the upper and lower limits of the signal as defined by the mapping file you selected to 'decode' this signal. Using these you can easily see whether a displayed signal is in or out of band.

 

To display barriers on the current graph:

  1. If using the Classic menus, select the Analyze menu
  2. Go to the Graph menu and click on Barrier.
  3. Barriers will appear on the graph view similar to Fig 1.2

Spectrum Graph showing +10MHz & -10MHz Barriers

 

Amplitude Settings

Classic Amplitude menu

 

Modern Amplitude menu

Amplitude settings adjust the vertical positioning of the graph on the screen.

 

Menu Item Action
Full Scale Sets the currently displayed spectrum's amplitude to maximum. Has the effect of removing any adjustments you made to the Scale setting below.
Ref Level Opens an on-screen keypad & slider that allows you to set the site specific power level calibration for the top bar of the power spectrum.
Scale Opens an on-screen keypad and slider that allows you to set the vertical values for the power spectrum on-screen grid

Frequency Settings

 

Classic Frequency menu

Modern Frequency menu

 

 

Menu Item Action
Center Opens a control window that allows you to enter a frequency for the center of the graph
Span Expands the width of the graph
Full Span

Returns the graph to full span. Has the effect of undoing any adjustments you might have made to the Span setting above.
VBW

Modifies the Video Bandwidth. Reducing the bandwidth ‘smooths’ the waveform and produces a thinner trace by decreasing the amount of noise. This is useful when identifying a weak interferer which would otherwise be lost in the noise.
RBW

Allows you to balance the amount of detail against the scan speed of the signal. Increasing the resolution bandwidth can help resolve a signal that appears noisy or takes time to settle. (See Fig 1.7 )
