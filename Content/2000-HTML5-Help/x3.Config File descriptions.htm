Configuration Files

 

Integris configuration files define the parameters for Radio Head testing. These parameters include the types of tests transmitted to the RRH as well as pass/fail values.

 

File Name Description
 Debug Short 'Debug' test for Local SFP, Physical Layer and CPRI Layer, but only counts to 1000 radio frames
 Default-LTE 'Standard' CPRI LTE. Tests the Local SFP, Physical layer and CPRI layer. Counts to 10,000 radio frames. No inventory polling. Checks for cable loop-back condition.
 Ericsson-LTE-Basic Contains test parameters for Ericsson configuration LTE. Tests the Local SFP, Physical layer and CPRI layer. Counts to 10,000 radio frames. No inventory polling. Checks for cable loop-back condition.
 Ericsson-LTE Contains test parameters for Ericsson configuration LTE. Tests the Local SFP, Physical layer, and CPRI layer. Counts to 10,000 radio frames. No inventory polling. Checks for cable loop-back condition.
 Lucent-LTE-Basic Contains test parameters for Alcatel-Lucent configuration LTE. Tests the Local SFP, Physical layer, and CPRI layer. Counts to 10,000 radio frames. No inventory polling. Checks for cable loop-back condition.
 Lucent-LTE-Turnup Intended as a turnkey test for new Alcatel-Lucent LTE installations. Tests the Local SFP, Physical and CPRI layers. Counts to 10,000 radio frames, plus reads radio head and remote SFP inventory. Returns radio head metrics: VSWR, Return Loss, and TX/RX power.
 Lucent-LTE Contains test parameters for Alcatel-Lucent configuration LTE. Tests the Local SFP, Physical layer and CPRI layer. Counts to 10,000 radio frames, plus reads radio head and remote SFP inventory.
 Selftest Used with a loopback connector at the end of a cable attached to Integris test port 1, to check the Integris 2000 system is working correctly
