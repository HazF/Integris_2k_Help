Working with Screen Profiles

Working with Screen Profiles

Location    
GUI or Keyboard Not applicable  
Classic menu System > Save Profile/Load Profile
Modern menu System > Save Profile/Load Profile

 

You can save the current test environment, then load it at a later date. This can be useful when detailing a set of tests that are required to be run in certain circumstances. A test plan to be run if the radio head fails the installation test, for example.

 

Saving a profile in this way saves:

  • All views and their locations
  • Status Pane settings
  • Test report complete with details and any test reports that have been run

Saving a Profile

Saving a Profile

To save a profile:

  1. Ensure the screen environment is set as you want
  2. Open the System menu and click on Save Profile (or press CTRL + Shift + S)
  3. A Save Profile dialog box will open allowing you to type a profile name or select a previously used one in order to overwrite it.
  4. Click on Ok

Loading a Saved Profile

Loading a Saved Profile

To load a profile that has been saved previously:

  1. Open the System menu and click on Load Profile (or press CTRL + Shift + L)
  2. A Load Profile dialog box will open allowing you to select from a list of saved profiles
  3. Select the desired profile from the list and click on Ok
