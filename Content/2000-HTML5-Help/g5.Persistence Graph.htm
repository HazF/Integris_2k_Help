Persistence Graph Configuration

Persistence graphs plot signal power magnitude against frequency, with colors denoting the percentage of time a plot point is placed at the current location

This type of graph is useful when looking for infrequent or intermittent signals.

10MHz swept signal seen on spectrum and persistence graphs

 

To open a persistence graph:

  1. If using the Classic menus, select the Analyze menu
  2. Go to the Graph menu and click on Type.
  3. Select the Persistent option (or in combination with another graph type)

Configuring the Persistent Graph

You can control the way the persistent graph represents data by either clicking on the live control pane, or right clicking. Both methods are shown below.

Persistent graph controls (persistance off)

 

Peak Hold maintains the graph's peak values until you disable it. This can display short period signals or those that occur very infrequently. When Peak Hold is disabled it appears as in Fig 1.2

To enable or disable Peak Hold, click the live icon on the persistence control panel. (See Fig 1.3 )

When Peak Hold is turned off the panel appears as in Fig 1.4 with the Decay setting disabled.

Persistent graph controls (persistance on)

 

 

10MHz swept signal showing persistence with Peak Hold on

 

Configure Colors: The range represented by the color spectrum can be modified to help reveal more detail.

To change the maximum value represented on the color scale:

  1. Click on the live color spectrum icon above the graph (See Fig 1.4 )
  2. Drag the slider with the mouse, or enter the desired value using the keypad, then click Enter (See Fig 1.6 )
  3. Click anywhere on the graph to close the slider/keypad

Persistence graph color scale setting

 

Decay: The live Decay icon is only displayed on the persistence graph when Peak Hold is disabled.

Clicking on the Decay icon opens a slider and keyboard that allows you to modify the time plotted values are held on the graph. The range of values is between 100 milli seconds and 1 second, and the shorter the period, the quicker displayed points are removed from the graph. (See Fig 1.7 )

Persistence Graph Right Click Options

By right clicking anywhere on the graph you can open a menu of options. (See Fig 1.8 )

 

Menu Item Meaning
Grid

Enable this option to superimpose a grid onto the graph
Type

Opens a menu displaying a range of graph types or combinations of graph types, to display
Close View Click to close the currently selected graph. See Using Multiple Views (Only displayed if multiple views are being displayed)
Move View Click to move the current graph on the screen in relation to other views. See Using Multiple Views (Only displayed if multiple views are being displayed)
