Testing an SFP

Location  
GUI or Keyboard F5 to run test
Classic menu Test > Select Test > Local SFP
Modern menu Test > Select Test > Local SFP

NOTE: Local SFP tests relate to the SFP fitted into the Integris test port(s). They should not be confused with the SFP information reported by the Install test. The SFP information returned during the Install test relates to the SFP connected into the RRH.

The Integris 2000 is able to read the vital product data (VPD) from an SFP located in either or both of its test ports. It can also provide basic functional testing, as well as providing metrics such as operating temperature and transmit & receive signal power.

  1. Open the Test menu and select NewT.
  2. Now click on RUN from the same menu (or press the F5 key),
 

Modern menu
 

SFP Results

When the test sequence has been completed you can optionally save the test results by adding them to a report. To add to an existing or new report file, click on the yellow Add To Report button.

You can now choose a name for the test results, or you can accept the default name based on the date and time of the test. Once you have finished editing the results name, click on OK.
