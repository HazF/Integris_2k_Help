How to Apply OCNS

The number of users on a radio system loads the resources available and affects how the system performs.

In order to simulate real-world loading, Integris 2000 includes the ability to apply Orthogonal Channel Noise Simulation (OCNS) to the last phase of the Install tests. This 'forward loads' the system and simulates user loading similar to those expected during in-service operation.

 

To switch on OCNS:

  1. Select the Test menu, and click on Select Test. Choose the Install option.

    Note: The Integris 2000 Status Pane shows different options and settings depending on what operation the application is set to perform.

    Only during the Install test does the Status Pane include controls to set transmitter power, insert OCNS and other transmitter related settings.

  2. Near the bottom of the Status Pane at the top left of the screen, click OCNS

  3. Select the OCNS value required to simulate the desired circuit loading. (See Fig 1.1 )
